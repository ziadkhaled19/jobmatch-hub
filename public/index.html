JobMatchHub API

A RESTful API for job matching platform with JWT authentication, role-based access control, and comprehensive CRUD operations for jobs and applications.

API Status

Authentication Endpoints

POST /api/auth/register - Register new user
POST /api/auth/login - Login user
GET /api/auth/me - Get current user

Job Endpoints

GET /api/jobs - Get all jobs with filters
POST /api/jobs - Create new job (Recruiter only)
GET /api/jobs/:id - Get single job

Application Endpoints

POST /api/applications - Apply for a job
GET /api/applications/my-applications - Get user's applications
GET /api/applications/job/:jobId - Get applications for a job

Features

  • JWT-based authentication with role-based access control
  • User roles: job_seeker, recruiter, admin
  • Complete CRUD operations for job postings
  • Advanced job filtering (location, salary, experience, etc.)
  • Job application system with status tracking
  • Email notifications for key events
  • Rate limiting and security middleware
  • MongoDB database with Mongoose ODM
  • Input validation and error handling
  • API documentation and health checks
